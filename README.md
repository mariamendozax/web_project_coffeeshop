# Triple Espresso Café

## Descripción del proyecto

Este proyecto consiste en la creación de una página web para el café **Triple Espresso**, desarrollada como parte del curso de HTML y CSS avanzados en el sprint 4.

La página presenta información clave del café, como sus horarios de atención. Un pequeño apartado con frames de recetas para acercarse al basto mundo del café de especialidad, dirección y enlaces a redes sociales. Todo esto siguiendo un diseño proporcionado en el brief del proyecto en sus diferentes fases. El objetivo principal fue replicar el diseño con precisión, aplicando buenas prácticas en html y css.

Para su desarrollo se utilizaron las siguientes tecnologías:

- **HTML** para la estructura semántica del sitio.
- **CSS** para el diseño visual, posicionamiento de elementos y estilos avanzados.
- Metodología **BEM** para la organización y nomenclatura de clases CSS.
- Uso de **Flexbox** y **position (relative y absolute)** para lograr la disposición correcta de los elementos según el diseño.
- Uso de **Pseudoclases** para lograr cierta interaccion a los diferentes estilos..

## Planes de mejora

En el futuro, me gustaría implementar las siguientes mejoras en el proyecto:

- Añadir **diseño responsivo** completo para dispositivos móviles y tablets mediante media queries.
- Incorporar **animaciones y transiciones CSS** para mejorar la experiencia del usuario.
- Agregar una sección interactiva de reservas utilizando **JavaScript**.
- Optimizar el rendimiento del sitio mediante la compresión de imágenes y mejoras en la estructura del CSS.

Estas mejoras permitirían que el proyecto evolucione hacia una experiencia más dinámica y cercana a un sitio web real en producción.
